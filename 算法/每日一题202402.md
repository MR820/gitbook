# Kadane 算法

## 71. [最大子数组和](https://leetcode.cn/problems/maximum-subarray/)

```go
func maxSubArray(nums []int) int {
	n := len(nums)
	max := nums[0]
	for i := 1; i < n; i++ {
		if nums[i-1]+nums[i] > nums[i] {
			nums[i] = nums[i-1] + nums[i]
		}
		if nums[i] > max {
			max = nums[i]
		}
	}
	return max
}

```
